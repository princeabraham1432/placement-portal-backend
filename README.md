# Placement Portal Backend

Spring Boot REST API for managing student records.

## Features
- Student CRUD APIs
- DTO-based request handling
- Global exception handling
- H2 in-memory database
- Clean layered architecture

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database

## APIs
- POST /students
- GET /students
- GET /students/{id}
- PUT /students/{id}
- DELETE /students/{id}
